# Thalia Interface Mood Board

## Color Palette

### Primary Colors
- **Deep Space Black**: `#0A0E17` - Base background color
- **Neural Blue**: `#00A6FF` - Primary accent color for active elements
- **Digital Magenta**: `#FF00A6` - Secondary accent for important highlights
- **Cyber Teal**: `#00FFD1` - Tertiary accent for system indicators

### Secondary Colors
- **Terminal Green**: `#00FF66` - Success states and positive indicators
- **Warning Amber**: `#FFCC00` - Caution states and notifications
- **Error Red**: `#FF3333` - Error states and critical alerts
- **Muted Slate**: `#7A8B9C` - Secondary information and inactive elements

### Gradient Combinations
- **Neural Gradient**: `linear-gradient(135deg, #00A6FF, #0A0E17)`
- **Cyberpunk Sunset**: `linear-gradient(135deg, #FF00A6, #FFCC00)`
- **Digital Horizon**: `linear-gradient(135deg, #00FFD1, #00A6FF)`
- **Deep Mind**: `linear-gradient(135deg, #0A0E17, #232838)`

## Typography

### Primary Font
- **Exo 2**: Modern, technical, and highly legible with futuristic characteristics
  - Weights: Light (200), Regular (400), Medium (500), Bold (700)
  - Use for main interface text, headings, and important information

### Secondary Font
- **JetBrains Mono**: Monospaced font for code-like elements and technical data
  - Weights: Regular (400), Medium (500)
  - Use for system information, metrics, and terminal-style displays

### Accent Font
- **Orbitron**: Ultra-futuristic display font
  - Weights: Medium (500), Bold (700)
  - Use sparingly for major headings and key interface elements

## Visual Elements

### Textures
- Subtle digital noise patterns
- Circuit board-inspired line work
- Data flow patterns (binary, hexadecimal)
- Topographic/wireframe surfaces

### UI Components
- Asymmetrical panels with glowing edges
- Holographic cards with variable transparency
- Terminal-inspired input fields
- Animated data visualizations
- Floating action buttons with particle effects

### Animation Styles
- Glitch transitions between major states
- Smooth, physics-based micro-interactions
- Pulsing/breathing effects for ambient indicators
- Electric/neon effects for active elements
- Particle systems for background ambience

## Interface Concepts

### Conversation Visualization
- Neural network-inspired conversation flows
- Voice waveforms that transform into text
- Dynamic typography that reflects emotional tone
- Thought bubbles that form and connect like nodes

### Memory System Visualization
- 3D memory palace with navigable spaces
- Time-based memory timeline with clustering
- Neural pathway strengthening for frequently accessed memories
- Fading/glowing effects for memory importance

### Emotional Expression
- Color shifts based on emotional state
- Particle density/movement reflecting current mood
- Abstract visualizations of PAD (Pleasure-Arousal-Dominance) emotional model
- Subtle background animations tied to emotional state

### System Status
- Cyberpunk-inspired metrics and gauges
- Circuit-like activity indicators
- Matrix-style data streams for background processing
- Holographic system health indicators
